# Bono Steel Customer Satisfaction Survey

A React/Vite customer satisfaction survey built from the supplied questionnaire.

## Local Development

```bash
npm install
npm run dev
```

## Production Build

```bash
npm run build
```

## Vercel

Import this folder as a Vercel project. Vercel will detect Vite automatically.

- Build command: `npm run build`
- Output directory: `dist`

The current submission flow downloads or emails a JSON response from the browser. Connect a form endpoint or database later if you want hosted response collection.
